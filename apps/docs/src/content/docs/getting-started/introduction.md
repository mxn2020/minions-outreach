# Minions Outreach

Email templates, drip sequences, personalization rules, and send schedules

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-outreach/sdk`) — Core library for Node.js
- **Python SDK** (`minions-outreach`) — Python bindings
- **CLI Tool** (`@minions-outreach/cli`) — Command-line interface
- **Web App** — Interactive playground at [outreach.minions.wtf](https://outreach.minions.wtf)
