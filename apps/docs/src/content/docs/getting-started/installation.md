---
title: Installation
description: Install Minions Outreach packages
---

## TypeScript / Node.js

```bash
npm install @minions-outreach/sdk minions-sdk
```

## Python

```bash
pip install minions-outreach
```

## CLI

```bash
npm install -g @minions-outreach/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
